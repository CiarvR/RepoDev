Projet react consistant à créer un site de drapeaux hébergeant une API. On peu y modifier le nombre de pays présents sur l'écran grâce à la molette, on peu séléctionner un continent pour y voir seulement les pays de ce continent, les données JSON de l'api nous disent : le nombre d'habitant du pays. Les pays sont aussi rangés par ordre d'habitant croissants.
